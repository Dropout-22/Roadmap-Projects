# 🎯 Number Guessing Game

A terminal-based Python game where intuition meets probability.  
Guess the hidden number across different difficulty levels, track your score, and challenge your own pattern recognition.

---

## 🚀 Features

- 🎚️ Multiple difficulty levels:
  - Easy (1–50, 10 guesses)
  - Medium (1–100, 5 guesses)
  - Hard (1–500, 3 guesses)

- ⏱️ Timer-based scoring system
- 🧠 Hint system (Higher / Lower)
- 🕶️ Cheat mode (risk + curiosity mechanic)
- 🏆 Score tracking across rounds
- 🔁 Replay system

Inspired by https://roadmap.sh/projects/number-guessing-game
